What are block and non block elements

What is a block level element?

A block level element is something that always starts on a
new line, the browsers automatically adds a margin before,
and after the element and it always takes the most width
possible.

What is a non-block element

A non-block element is the opposite of a block element,
it does not always start on a new line and it only takes
as much width as needed.
